# fibonacci

## Instructions

Write a program that calculates the nth number in Fibonacci series.

As an extra challenge, try to optimize your solution so that you can find
the 1000th number in the Fibonacci Series in under a second.

## Example

```
fibonacci(6)
=> 8

fibonacci(8)
=> 21

fibonacci(13)
=> 233
```
